# Document
无论是文档还是其他资源，都会转成一个 document 对象。
在的包在 DocumentLoader 下面，有的则在 community 社区下面。
根据不同的类型，选择不同的资源解析包（参考官方文档）。

## 本地资源

- 文本 Loader
需要先安装 文本Loader 包
- PDF Loader
需要安装 PDSLoader 包，在 community 社区的库下面

## web 资源

- 网页 Loader
根据 ReplLiader 抓取网页内容，并整理成 Document
- Github Loader
GithubRepoLoader 抓取 Github 资源，使用
